# Tabs Sort Manager

Chrome 확장 프로그램으로, 사이드 패널에서 탭을 효율적으로 관리할 수 있는 도구입니다.

## 주요 기능

### 📋 탭 그룹 관리
- 탭 그룹을 시각적으로 확인하고 관리
- 그룹별로 탭을 접기/펼치기
- 그룹 헤더로 드래그하여 탭을 그룹에 추가
- 그룹 해제(Ungroup) 기능

### 🌐 도메인별 정렬
- 그룹되지 않은 탭을 도메인별로 자동 분류
- 도메인별 섹션 접기/펼치기
- 도메인별 자동 정렬 기능

### 🔍 검색 기능
- 탭 제목, URL, 도메인, 그룹명으로 검색
- 실시간 필터링

### 🎨 테마 지원
- 다크/라이트 테마 전환
- 테마 설정 자동 저장

### 🖱️ 드래그 앤 드롭
- 탭을 드래그하여 순서 변경
- 탭을 그룹 헤더로 드래그하여 그룹에 추가
- 탭 간 드래그로 재정렬

### ✅ 탭 선택 및 그룹 생성
- 개별 탭 선택 (체크박스)
- 도메인 섹션 전체 선택
- 선택한 탭으로 새 그룹 생성
- 그룹 색상 선택 가능

## 설치 방법

1. Chrome 브라우저에서 `chrome://extensions/` 접속
2. 우측 상단의 "개발자 모드" 활성화
3. "압축해제된 확장 프로그램을 로드합니다" 클릭
4. `tabs-sort-manager` 폴더 선택

## 사용 방법

### 기본 사용
1. 확장 프로그램 아이콘 클릭하여 사이드 패널 열기
2. 현재 창의 모든 탭이 표시됩니다

### 탭 그룹 생성
1. 그룹에 포함시킬 탭들을 체크박스로 선택
2. "선택 탭 그룹 생성" 버튼 클릭
3. 그룹 이름 입력 및 색상 선택

### 탭 정렬
- **수동 정렬**: 탭을 드래그하여 원하는 위치로 이동
- **자동 정렬**: "도메인별 자동정렬" 버튼으로 그룹되지 않은 탭을 도메인별로 정렬

### 검색
- 상단 검색창에 키워드 입력
- 탭 제목, URL, 도메인, 그룹명에서 검색

### 그룹 관리
- 그룹 헤더의 "Ungroup" 버튼으로 그룹 해제
- 그룹 개요 섹션에서도 그룹 해제 가능
- 그룹 칩 클릭 시 해당 그룹 섹션으로 스크롤

## 파일 구조

```
tabs-sort-manager/
├── manifest.json      # 확장 프로그램 설정
├── sidepanel.html     # 사이드 패널 HTML
├── sidepanel.css      # 스타일시트
├── sidepanel.js       # 메인 로직
└── sw.js              # Service Worker
```

## 기술 스택

- **Manifest V3**: 최신 Chrome 확장 프로그램 API
- **Vanilla JavaScript**: 프레임워크 없이 순수 JavaScript 사용
- **Chrome APIs**:
  - `chrome.tabs`: 탭 관리
  - `chrome.tabGroups`: 탭 그룹 관리
  - `chrome.sidePanel`: 사이드 패널
  - `chrome.storage`: 설정 저장

## 권한

- `sidePanel`: 사이드 패널 사용
- `tabs`: 탭 정보 접근 및 관리
- `tabGroups`: 탭 그룹 관리
- `storage`: 테마 설정 저장
