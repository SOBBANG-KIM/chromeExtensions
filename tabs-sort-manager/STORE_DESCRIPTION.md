## 자세한 설명 (Detailed Description)

### 한국어 버전

```
📋 탭 관리가 이렇게 쉬웠나요?

Tabs Sort Manager는 Chrome 사이드 패널에서 모든 탭을 한눈에 보고 관리할 수 있는 강력한 도구입니다. 수많은 탭으로 인해 혼란스러웠던 브라우징 경험을 깔끔하게 정리해드립니다.

✨ 주요 기능

🔍 스마트 검색
• 탭 제목, URL, 도메인, 그룹명으로 실시간 검색
• 원하는 탭을 빠르게 찾아 이동

🌐 도메인별 자동 정렬
• 그룹되지 않은 탭을 도메인별로 자동 분류
• 한 번의 클릭으로 관련 탭들을 모아서 정리
• 도메인별 섹션을 접고 펼쳐서 깔끔하게 관리

📁 탭 그룹 관리
• 여러 탭을 선택하여 새 그룹 생성
• 9가지 색상으로 그룹 구분
• 그룹 헤더로 드래그하여 탭 추가
• 그룹 개요에서 한눈에 모든 그룹 확인
• 그룹 해제 기능으로 유연한 관리

🖱️ 직관적인 드래그 앤 드롭
• 탭을 드래그하여 원하는 순서로 재배치
• 그룹 헤더에 드롭하여 그룹에 추가
• 탭 간 드래그로 빠른 정렬

✅ 스마트 선택 기능
• 개별 탭 체크박스로 선택
• 도메인 섹션 전체 선택 지원
• 선택한 탭으로 한 번에 그룹 생성

🎨 테마 지원
• 다크/라이트 테마 전환
• 선호하는 테마로 설정 가능
• 테마 설정 자동 저장

💡 사용 방법

1. 확장 프로그램 아이콘을 클릭하여 사이드 패널을 엽니다.
2. 현재 창의 모든 탭이 표시됩니다.
3. 검색창에 키워드를 입력하여 원하는 탭을 찾습니다.
4. 탭을 드래그하여 순서를 변경하거나 그룹에 추가합니다.
5. 여러 탭을 선택하고 "선택 탭 그룹 생성" 버튼으로 그룹을 만듭니다.
6. "도메인별 자동정렬" 버튼으로 관련 탭들을 자동으로 정리합니다.

🔒 개인정보 보호

이 확장 프로그램은:
• 모든 데이터를 로컬에만 저장합니다
• 외부 서버로 정보를 전송하지 않습니다
• 탭 정보는 현재 브라우저 창에서만 사용됩니다

📌 팁

• 그룹 개요 섹션의 그룹 칩을 클릭하면 해당 그룹으로 바로 이동합니다.
• 도메인 섹션의 "전체" 체크박스로 해당 도메인의 모든 탭을 한 번에 선택할 수 있습니다.
• 그룹 헤더나 그룹 개요에서 "Ungroup" 버튼으로 그룹을 해제할 수 있습니다.
• 검색 중에는 매칭되는 탭만 표시되어 더욱 깔끔하게 관리할 수 있습니다.

이제 수많은 탭 속에서 길을 잃지 마세요. Tabs Sort Manager로 브라우징을 더욱 효율적으로 만들어보세요!
```

### English Version

```
📋 Manage Your Tabs with Ease

Tabs Sort Manager is a powerful tool that lets you view and manage all your tabs in Chrome's side panel. Transform your cluttered browsing experience into an organized workspace.

✨ Key Features

🔍 Smart Search
• Real-time search across tab titles, URLs, domains, and group names
• Quickly find and navigate to any tab

🌐 Domain-Based Auto Sort
• Automatically categorize ungrouped tabs by domain
• One-click organization of related tabs
• Collapsible domain sections for clean management

📁 Tab Group Management
• Select multiple tabs to create new groups
• 9 color options for group identification
• Drag tabs to group headers to add them
• Overview section to see all groups at a glance
• Ungroup functionality for flexible management

🖱️ Intuitive Drag & Drop
• Drag tabs to rearrange them in any order
• Drop tabs on group headers to add to groups
• Quick sorting by dragging between tabs

✅ Smart Selection
• Individual tab selection with checkboxes
• Bulk selection for entire domain sections
• Create groups from selected tabs in one click

🎨 Theme Support
• Dark/Light theme toggle
• Set your preferred theme
• Theme settings automatically saved

💡 How to Use

1. Click the extension icon to open the side panel.
2. All tabs in the current window are displayed.
3. Type keywords in the search box to find tabs.
4. Drag tabs to reorder or add to groups.
5. Select multiple tabs and click "Create Group" to organize them.
6. Use "Sort by Domain" to automatically organize related tabs.

🔒 Privacy

This extension:
• Stores all data locally only
• Does not send information to external servers
• Uses tab information only within the current browser window

📌 Tips

• Click group chips in the overview section to jump to that group.
• Use the "Select All" checkbox in domain sections to select all tabs at once.
• Use the "Ungroup" button in group headers or overview to ungroup tabs.
• During search, only matching tabs are displayed for cleaner management.

Stop getting lost in countless tabs. Make your browsing more efficient with Tabs Sort Manager!
```
