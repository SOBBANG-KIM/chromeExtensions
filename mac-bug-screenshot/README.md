# mac-bug-screenshot

스토어 모드(네이티브 없음)와 네이티브 모드를 전환할 수 있습니다.

## 크롬 확장 팝업(UI)

1) `mac-bug-screenshot/extension`을 크롬 확장으로 로드
2) 팝업에서 **캡처 모드** 선택 (스토어/네이티브)
3) **전체 화면 캡처** 버튼 클릭

네이티브 모드를 쓰려면 먼저 아래를 설치하세요.
```bash
npm install -g @mac-bug-screenshot/native-host
mac-bug-screenshot-install <확장_프로그램_ID>
```

## 파일명/경로 설정

스토어 모드에서는 **파일명 접두사**만 적용됩니다.
네이티브 모드에서는 **저장 경로**도 적용됩니다.

## 참고

- 스토어 모드: 화면 선택 동의 필요, 저장 위치는 다운로드 설정/다이얼로그, 미리보기는 새 탭.
- 네이티브 모드: npm 패키지 설치 후 사용.
