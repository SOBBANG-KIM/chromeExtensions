# Mate Jira Quick Add

Jira 이슈 등록, Worklog 기록, 내 이슈 조회를 빠르게 처리하는 Chrome 확장 프로그램입니다. 로그인된 Jira 세션을 그대로 사용하고, Google Calendar 회의를 불러와 Worklog를 한 번에 또는 회의별로 저장할 수 있습니다.

## Chrome 웹스토어용 설명

**Mate Jira Quick Add**는 Jira 이슈 생성, Worklog 입력, 내 이슈 목록 조회를 한 곳에서 처리하는 확장 프로그램입니다.  
로그인된 Jira 세션을 그대로 사용하며, Google Calendar 회의를 불러와 날짜/시간 범위 기준으로 Worklog를 자동 작성할 수 있습니다.

**주요 기능**
- **이슈 등록**: 프로젝트 키/이슈 타입/요약/설명 입력 후 즉시 생성
- **부작업 생성**: 상위 이슈 키 입력 시 부작업 생성 지원
- **동적 필수 필드**: 프로젝트/이슈 타입에 따라 Jira에서 요구하는 필수 항목을 자동으로 조회해 표시
- **구성요소 선택**: 프로젝트에 구성요소가 있으면 전용 셀렉터로 선택
- **프로젝트 키 즐겨찾기**: 자주 쓰는 프로젝트 키를 자동완성 목록으로 기억
- **Worklog 등록**: 이슈 키/시작 시각/작업 시간/설명 입력으로 기록
- **Google Calendar 연동**: 회의 목록·시간 자동 입력, 회의별로 개별 Worklog 저장
- **내 이슈 목록**: 상태·프로젝트 필터로 할당 이슈 조회, 페이지네이션, 목록에서 바로 Worklog 등록
- **로그인 상태 표시**: Jira 로그인 상태는 상단에서, Google 로그인 상태는 add worklog 탭에서 확인

## 사용 가이드

### 1) add jira — 이슈 등록
1. `add jira` 탭에서 `프로젝트 키`, `이슈 타입`, `요약`, `설명` 입력
2. 이슈 타입이 `부작업`이면 `상위 이슈 키` 입력
3. 프로젝트에 구성요소가 있으면 `구성요소` 선택, 프로젝트/이슈 타입에 필수 항목이 있으면 자동 표시되는 필드에 입력
4. `등록` 클릭 → 성공 시 이슈 키 표시 (마지막 이슈 키는 Worklog 탭에서 사용 가능)

### 2) add worklog — Worklog 등록

#### 기본 입력으로 저장
1. `add worklog` 탭에서 `이슈 키` 입력 (또는 `마지막 이슈 키 사용` 클릭)
2. `시작 시각`, `작업 시간`, `설명` 입력
3. `한번에 저장` 클릭 → 하나의 Worklog로 저장

#### Google Calendar 회의 불러오기
1. `Google 로그인` 클릭 후 인증 (로그인 성공 시 해당 버튼은 숨겨짐)
2. `날짜`와 `시간 범위` 입력 (예: `09:30 ~ 18:30`)
3. `회의 불러오기` 클릭  
   - **회의가 있을 때**: 설명에 회의 목록, 작업 시간에 회의 시간 합계 자동 입력. `N개 회의 불러옴` 표시
   - **회의가 없을 때**: 설명 `회의없음`, 작업 시간 `8h`로 설정
4. 저장 방식 선택  
   - **한번에 저장**: 현재 입력(시작 시각/작업 시간/설명)을 하나의 Worklog로 저장  
   - **회의별로 저장**: 캘린더에서 불러온 각 회의마다 **시작 시각·작업 시간**을 나누어 개별 Worklog로 저장 (회의 0개일 때는 버튼 비활성화)

### 3) add my issues — 내 이슈 목록
1. `add my issues` 탭에서 `상태` 입력 (기본값: 진행 중)
2. `목록 불러오기` 클릭 → 해당 상태의 내 이슈가 속한 프로젝트 목록이 `프로젝트` 필터에 채워짐
3. 필요 시 `프로젝트` 필터로 좁혀서 조회. 필요 시 `이전`/`다음`으로 페이지 이동
4. 이슈 제목 클릭 시 Jira 이슈 페이지로 이동
5. 항목의 `작업로그` 버튼 클릭 시 목록에서 바로 시작 시각/소요 시간/설명을 입력해 Worklog 저장 가능

## 입력 형식
- **작업 시간**: Jira 형식 (`1h`, `30m`, `1h 30m` 등)
- **시작 시각**: 로컬 시간 기준 `datetime-local`
- **시간 범위**: `09:30 ~ 18:30` 형식 (공백·`~`·`-` 사용 가능)

## 고정 API 정보
- Base URL: `https://jira.foodtechkorea.com/rest`
- API 버전: `v2`

## 권한 안내
- `storage`: 입력값·설정·드래프트 저장
- `identity`: Google OAuth 인증
- `host_permissions`: Jira/Google API 호출
