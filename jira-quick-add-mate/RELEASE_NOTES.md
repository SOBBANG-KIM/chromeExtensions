# 릴리즈 노트

## v1.2.4

### ✨ 새로운 기능

#### 이슈 등록 시 Worklog 바로 기재 (add jira)
- 이슈를 등록할 때 **시작 시각·작업 시간**을 함께 입력하면 worklog까지 한 번에 저장됩니다 (선택 사항).
- 시작 시각은 현재 시각으로 자동 채워집니다.

#### 내 이슈 목록을 3개 서브탭으로 개편 (add my issues)
- **내 이슈 목록**: 상태·프로젝트로 필터링. 상태 자동완성(BACKLOG/할 일/진행 중/개발 완료/IN REVIEW/완료) 및 초기화 버튼 추가. 상태를 비우면 모든 상태를 조회합니다.
- **미기록 티켓**: 선택한 달에 상태가 바뀌었거나 새로 생성된 티켓 중, 그 달에 worklog가 하나도 없는 것만 모아 보여줍니다. 월 선택·프로젝트 필터 지원.
- **장기 진행**: 현재 "진행 중" 상태인 티켓 중, 최근 전환일로부터 오래 머문 티켓을 기준일 이상으로 필터링해 보여줍니다.
- 각 티켓의 **작업로그** 토글을 열면 기존 worklog 이력과 이번 달 로그 시간 뱃지가 함께 표시되고, 그 자리에서 바로 worklog를 저장할 수 있습니다.
- 각 탭에 **"ⓘ 조회 조건"** 클릭 툴팁으로 조회 기준을 안내합니다.

#### 월 현황 worklog 탭 신설
- 선택한 달의 **일/주/월 단위 worklog 집계**를 보여주고, 하루 목표(8시간) 대비 **부족·초과·충족**을 색상으로 표시합니다.
- 부족한 날은 "채우기"로 바로 worklog 입력 화면으로 이동합니다.

### 🔧 개선
- 팝업 타이틀 옆에 **현재 버전 표기** 추가 (v1.2.4).
- worklog 시간 표기를 Jira 근무 설정(1일 8시간) 기준 `Nd Xh Ym` 형식으로 통일.
